## How Aether Works in OpenCode

OpenCode uses slash commands as the primary interface for colony operations:

```
/ant-init "Build feature X"     # Start a colony
/ant-plan                       # Generate project phases
/ant-build <phase>              # Execute a phase
/ant-continue                   # Verify and advance
/ant-status                     # Colony dashboard
/ant-run                        # Autopilot mode
```

Agent definitions live in `.opencode/agents/*.md` (Markdown with YAML frontmatter).

## Key Commands

### Setup and Getting Started

| Command | Purpose |
|---------|---------|
| `/ant-init "<goal>"` | Start a colony with a goal |
| `/ant-colonize` | Analyze existing codebase |
| `/ant-plan` | Generate project phases |
| `/ant-build <N>` | Execute a phase |
| `/ant-continue` | Verify, learn, advance |
| `/ant-status` | Colony dashboard |
| `/ant-run` | Autopilot all phases |

### Pheromone Signals

| Command | Priority | Purpose |
|---------|----------|---------|
| `/ant-focus "<area>"` | normal | Guide colony attention |
| `/ant-redirect "<pattern>"` | high | Hard constraint -- avoid this |
| `/ant-feedback "<note>"` | low | Gentle adjustment |
| `/ant-pheromones` | -- | View active signals |

### Session Management

| Command | Purpose |
|---------|---------|
| `/ant-resume` | Quick session restore |
| `/ant-resume-colony` | Full recovery view |
| `/ant-pause-colony` | Save state and create handoff |

### Lifecycle

| Command | Purpose |
|---------|---------|
| `/ant-seal` | Seal completed colony |
| `/ant-entomb` | Archive completed colony |
| `/ant-update` | Update system files from hub |

## Typical Workflow

```
/ant-init "Build feature X"       # Start colony
/ant-colonize                      # Analyze existing code (optional)
/ant-plan                          # Generate phases
/ant-focus "security"              # Guide colony (optional)
/ant-build 1                       # Execute phase 1
/ant-continue                      # Verify and advance
/ant-run                           # Or autopilot all phases

# After completing:
/ant-seal                          # Mark colony complete
/ant-entomb                        # Archive
```
