    Flags: `--branch` (string, required — the worktree branch to merge back)

    **Logic flow (per D-04, D-05, D-06, D-13):**

    1. **Validate input**: Get --branch flag. If empty, outputError and return.

    2. **Load state**: Load COLONY_STATE.json. Find the WorktreeEntry with matching branch name.
       - If not found: `outputError(1, "worktree {branch} not found in state tracking", nil)`, return.
       - If found with status "merged": `outputError(1, "worktree {branch} already merged", nil)`, return.
       - If found with status "orphaned": proceed (allow re-merge of orphaned worktrees per state transition D-08: orphaned -> merged).

    3. **Gate 1: Run tests** (per D-04):
       ```go
       testCtx, testCancel := context.WithTimeout(context.Background(), BuildTimeout)
       defer testCancel()
       testCmd := exec.CommandContext(testCtx, "go", "test", "./...")
       testCmd.Dir = entry.Path  // CRITICAL: run in worktree directory (Pitfall 3)
       testOutput, testErr := testCmd.CombinedOutput()
       ```
       - If testErr != nil (non-zero exit): Create blocker (per D-05):
         ```go
         // Create blocker flag
         // Load pending-decisions.json or flags.json
         // Append new FlagEntry with Type="blocker", Description="Merge blocked: tests failed for {branch}", Source="worktree-merge-back"
         // Save via store.SaveJSON
         ```
         Then: `outputError(2, "merge blocked: tests failed for {branch}: {testOutput}", nil)`, return.

    4. **Gate 2: Clash check** (per D-04):
       - Run `git worktree list --porcelain` to get all worktree paths.
       - For each file modified in the worktree branch (run `git -C {entry.Path} diff --name-only main`), check if that file is also modified in any OTHER worktree.
       - If clashes found: Create blocker flag (same pattern as test failure), outputError with clash details, return.

       **Implementation**: Extract a helper function `checkClashesForWorktree(entryPath string, branch string) ([]string, error)` that returns a list of clashing file paths. This can reuse `parseWorktreePaths` from cmd/clash.go (same package, accessible directly).

    5. **Execute merge** (both gates passed):
       ```go
       // Checkout main in main worktree
       exec.CommandContext(gitCtx, "git", "checkout", "main").Run()
       // Merge the branch
       mergeOut, mergeErr := exec.CommandContext(gitCtx, "git", "merge", entry.Branch).CombinedOutput()
       ```
       - If merge fails: Create blocker, outputError, return.

    6. **Auto-cleanup** (per D-06):
       ```go
       // Step 6a: Remove worktree directory
       exec.CommandContext(pruneCtx, "git", "worktree", "remove", entry.Path, "--force").CombinedOutput()
       // Step 6b: Prune stale references
       exec.CommandContext(pruneCtx, "git", "worktree", "prune").Run()
       // Step 6c: Delete branch (tolerate "not found" error for fast-forward merges — Pitfall 6)
       err := exec.CommandContext(gitCtx, "git", "branch", "-d", entry.Branch).Run()
       if err != nil {
           // Check if it's a "not found" error — treat as success
           // Any other error: log warning but don't fail (branch cleanup is best-effort)
       }
       ```

    7. **Update state**:
       - Set entry.Status = WorktreeMerged
       - Set entry.UpdatedAt = time.Now().UTC().Format(time.RFC3339)
       - Save COLONY_STATE.json via store.SaveJSON

    8. **Audit log**:
       ```go

           "action": "worktree-merge",
           "branch": entry.Branch,
           "path":   entry.Path,
           "at":     time.Now().UTC().Format(time.RFC3339),
       })
       ```

    9. **Output success**:
       ```go
       outputOK(map[string]interface{}{
           "merged":    true,
           "branch":    entry.Branch,
           "worktree":  entry.Path,
           "status":    "merged",
       })
       ```

    func createBlocker(store *storage.Store, description string, source string) error {
        // Load existing flags
        var ff colony.FlagsFile
        if err := store.LoadJSON("pending-decisions.json", &ff); err != nil {
            if err2 := store.LoadJSON("flags.json", &ff); err2 != nil {
                ff = colony.FlagsFile{Decisions: []colony.FlagEntry{}}
            }
        }
        if ff.Decisions == nil {
            ff.Decisions = []colony.FlagEntry{}
        }
        ff.Decisions = append(ff.Decisions, colony.FlagEntry{
            ID:          generateFlagID(),
            Type:        "blocker",
            Description: description,
            Source:      source,
            CreatedAt:   time.Now().UTC().Format(time.RFC3339),
            Resolved:    false,
        })
        return store.SaveJSON("pending-decisions.json", ff)
    }

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Worktree directory -> go test | Untrusted code in worktree runs during test gate |
| Merge result -> main branch | Code crossing into main must pass both gates |
| Flag/blocker creation | Merge failures create persistent state (blocker flags) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-06 | Tampering | worktree-merge-back test gate | mitigate | Run `go test ./...` with BuildTimeout (120s); cmd.Dir set to worktree path to test the correct code (Pitfall 3) |
| T-06-07 | Tampering | worktree-merge-back clash gate | mitigate | Reuse parseWorktreePaths + git diff to detect file conflicts before merge |
| T-06-08 | Denial of Service | Long-running tests blocking merge | mitigate | BuildTimeout (120s) caps test execution; context.DeadlineExceeded produces clear error |
| T-06-09 | Elevation of Privilege | Auto-cleanup deleting wrong directory | mitigate | Cleanup only targets the specific worktree path from the state entry; path is constructed from sanitized branch name via sanitizeBranchPath |
</threat_model>
