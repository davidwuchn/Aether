2. SHARED E2E HELPER: e2eRecoverSetup
A helper function that all 10 tests call as their first action. It calls saveGlobals(t), resetRootCmd(t), then initRecoverTestStore(t) to get the store and dataDir. It also creates a bytes.Buffer, assigns it to stdout, and returns (store, dataDir, buf). The store is needed so loadActiveColonyState() succeeds.

```go
func e2eRecoverSetup(t *testing.T) (*storage.Store, string, *bytes.Buffer) {
    t.Helper()
    saveGlobals(t)
    resetRootCmd(t)
    s, dataDir := initRecoverTestStore(t)
    var buf bytes.Buffer
    stdout = &buf
    return s, dataDir, &buf
}
```

3. SHARED E2E HELPER: e2eRunRecover
Runs rootCmd with the given args, returns the error. Resets rootCmd args in cleanup.

```go
func e2eRunRecover(t *testing.T, args ...string) error {
    t.Helper()
    rootCmd.SetArgs(args)
    return rootCmd.Execute()
}
```

4. SHARED E2E HELPER: parseRecoverJSON
Parses stdout buffer as recoverJSONOutput. Fails the test if JSON is invalid.

```go
func parseRecoverJSON(t *testing.T, buf *bytes.Buffer) recoverJSONOutput {
    t.Helper()
    var output recoverJSONOutput
    if err := json.Unmarshal(buf.Bytes(), &output); err != nil {
        t.Fatalf("invalid JSON output: %v\nraw: %s", err, buf.String())
    }
    return output
}
```
