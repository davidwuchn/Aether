# Plan 01: Publish Command and Version Check

## Objective

Create the `aether publish` command that atomically syncs the binary and hub to the same version. Add `aether version --check` for manual downstream verification.

---

## Context

The current ad-hoc publish path is `aether install --package-dir "$PWD"`. This works but:
1. It is not discoverable — the command is called "install" not "publish"
2. It does not verify version agreement after completion
3. It led to the 1.0.19/1.0.20 mismatch where the hub was not updated to match the binary

The new `aether publish` command will:
- Wrap the existing hub sync logic from `install --package-dir`
- Build the binary with correct ldflags
- Verify that binary version and hub version agree after publish
- Fail loudly if they do not

The `aether version --check` flag will:
- Compare binary version against hub version.json
- Return non-zero exit code if they disagree
- Print a human-readable report of the mismatch

### action
Create `cmd/publish_cmd.go` with the following structure:

1. **Command definition** (follow installCmd pattern exactly):
   ```go
   var publishCmd = &cobra.Command{
       Use:   "publish",
       Short: "Publish Aether from source to the shared hub",
       Long:  "Build the aether binary and sync companion files to the hub, ...",
       Args:  cobra.NoArgs,
       RunE:  runPublish,
   }
   ```

2. **Flags** (in init()):
   - `--package-dir` (string, default "") — source directory, defaults to current directory
   - `--home-dir` (string, default "") — user home directory
   - `--channel` (string, default "") — stable or dev
   - `--binary-dest` (string, default "") — where to put the built binary
   - `--skip-build-binary` (bool, default false) — skip go build, use existing binary

3. **runPublish** function flow:
   a. Resolve channel via `runtimeChannelFromFlag(cmd.Flags())`
   b. Resolve packageDir (default to CWD if empty)
   c. Resolve homeDir (same logic as runInstall: HOME → USERPROFILE → error)
   d. Verify this is an Aether source checkout via `isAetherSourceCheckout(packageDir)` — return error if not
   e. Resolve version via `resolveVersion(packageDir)`
   f. Build binary (unless --skip-build-binary):
      - Call `buildLocalBinary(sourceRoot, destDir, version, channel)`
      - destDir comes from --binary-dest or `defaultLocalBinaryDest(homeDir, channel)`
   g. Sync hub (reuse `setupInstallHub` pattern):
      - hubDir = `filepath.Join(homeDir, defaultHubDirName(channel), "system")`
      - packageDirAether = `filepath.Join(packageDir, ".aether")`
      - Call `syncDirToHub(packageDirAether, hubDir)`
      - Write version.json to hubDir: `{"version":"<version>","updated_at":"now"}`
   h. **Verification** (NEW — this is the core requirement):
      - Read binary version: `resolveVersion()` on the built binary
      - Read hub version: `readInstalledHubVersion()` (which reads hub version.json)
      - If they differ, return `fmt.Errorf("publish verification failed: binary version %s does not match hub version %s", binaryVersion, hubVersion)`
      - If hub version was updated from an older version, print warning: `fmt.Fprintf(os.Stderr, "Warning: hub version updated from %s to %s\n", oldHubVersion, version)`
   i. Output success via `outputWorkflow()` with visual render

4. **Reusable functions** — do NOT duplicate; import/call existing:
   - `resolveVersion()` from root.go
   - `buildLocalBinary()` from install_cmd.go (line 886)
   - `syncDirToHub()` from install_cmd.go (line 660)
   - `isAetherSourceCheckout()` from install_cmd.go (line 832)
   - `defaultLocalBinaryDest()` from install_cmd.go (line 872)
   - `runtimeChannelFromFlag()` from runtime_channel.go
   - `defaultHubDirName()` from runtime_channel.go

5. **Helper: readHubVersionAtPath(hubDir string) string** — reads version.json at a specific path (extracted from readInstalledHubVersion which hardcodes resolveHubPath)

2. Modify RunE function:
   ```go
   RunE: func(cmd *cobra.Command, args []string) error {
       check, _ := cmd.Flags().GetBool("check")
       binaryVersion := resolveVersion()

       if check {
           hubVersion := readInstalledHubVersion()
           if hubVersion == "" {
               return fmt.Errorf("version check failed: no hub version found (is Aether installed?)")
           }
           if binaryVersion != hubVersion {
               return fmt.Errorf("version mismatch: binary=%s hub=%s", binaryVersion, hubVersion)
           }
           outputWorkflow(map[string]interface{}{
               "ok":      true,
               "message": fmt.Sprintf("Version check passed: binary and hub both at %s", binaryVersion),
               "version": binaryVersion,
           }, nil)
           return nil
       }

       outputOK(binaryVersion)
       return nil
   },
   ```
