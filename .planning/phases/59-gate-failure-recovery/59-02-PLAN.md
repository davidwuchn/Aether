CLI Command Contract (agreed with Plan 01):
```
aether gate-results-read
  → outputs JSON array of GateResultEntry to stdout
  → outputs [] if no prior results
  → no flags required

aether gate-results-write --name <string> --passed <bool> [--detail <string>]
  → replaces all gate_results with a single entry
  → --name is required (error if missing)
  → --passed is required
  → --detail is optional
  → outputs {"ok":true,"entry":{...}} on success
  → timestamp auto-generated as UTC RFC3339

aether should-skip-gate --name <string>
  → outputs "true" or "false" to stdout (literal string)
  → --name is required
  → always "false" for "tests_pass" regardless of prior result

aether gate-recovery-template --name <string>
  → outputs recovery template text to stdout
  → --name is required
  → outputs "No specific recovery instructions available for this gate." if unknown
```

From pkg/colony/colony.go (created in Plan 01):
```go
type GateResultEntry struct {
    Name      string `json:"name"`
    Passed    bool   `json:"passed"`
    Timestamp string `json:"timestamp"`
    Detail    string `json:"detail,omitempty"`
}
```
