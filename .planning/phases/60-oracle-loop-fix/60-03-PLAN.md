1. Create `scoreQuestionImpact(q oracleQuestion, plan oraclePlanFile, state oracleStateFile) float64`:
   The scoring function uses four weighted factors:
   - **Gap overlap (weight 0.35):** Count how many strings in `state.OpenGaps` share keywords with the question's text (lowercased word intersection, 3+ char words). More gap overlap = higher score.
   - **Contradiction overlap (weight 0.25):** Count how many strings in `state.Contradictions` share keywords with the question's text. Same word-intersection approach.
   - **Cross-question benefit (weight 0.20):** Count how many OTHER unanswered questions share keywords with this question. Researching this question likely benefits those others too.
   - **Confidence deficit (weight 0.20):** `(state.TargetConfidence - q.Confidence) / 100.0`. Larger deficit = higher score.

   Return the weighted sum. Clamp to [0, 1].

   Word matching: extract words by splitting on whitespace/punctuation, lowercase, filter to 3+ characters.

2. Create `selectOracleQuestionSmart(plan oraclePlanFile, state oracleStateFile) oracleQuestion`:
   ```go
   func selectOracleQuestionSmart(plan oraclePlanFile, state oracleStateFile) oracleQuestion {
       // Edge case: no questions
       if len(plan.Questions) == 0 {
           return oracleQuestion{ID: fmt.Sprintf("iteration-%d", state.Iteration), Text: "No oracle questions are currently available."}
       }

       var best oracleQuestion
       bestScore := -1.0
       anyUnanswered := false
       for _, q := range plan.Questions {
           if strings.EqualFold(strings.TrimSpace(q.Status), "answered") {
               continue
           }
           anyUnanswered = true
           score := scoreQuestionImpact(q, plan, state)
           // Priority boost for untouched questions
           if len(q.IterationsTouched) == 0 {
               score += 0.1
           }
           if score > bestScore {
               bestScore = score
               best = q
           }
       }
       if anyUnanswered {
           return best
       }
       // All answered -- return indicator
       return oracleQuestion{
           ID:   fmt.Sprintf("iteration-%d", state.Iteration),
           Text: "All oracle questions have been answered.",
       }
   }
   ```
