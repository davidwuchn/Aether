    **Per D-01 and D-02:** Create a table-driven smoke test suite.

    Create cmd/smoke_test.go with a single test function TestSmokeCommands that:

    1. Iterates over all registered subcommands via rootCmd.Commands()
    2. For each command, runs it in an isolated temp directory with no colony state
    3. Verifies the command does NOT panic (use recover() to catch panics)
    4. Verifies the command produces SOME output (stdout or stderr is non-empty) or returns a non-nil error (which is acceptable)
    5. Logs the command name and result for debugging

    Structure:
    ```go
    func TestSmokeCommands(t *testing.T) {
        commands := rootCmd.Commands()
        for _, cmd := range commands {
            cmd := cmd // capture range variable
            t.Run(cmd.Name(), func(t *testing.T) {
                saveGlobals(t)
                resetRootCmd(t)

                // Capture output
                var outBuf, errBuf bytes.Buffer
                stdout = &outBuf
                stderr = &errBuf
                defer func() {
                    stdout = os.Stdout
                    stderr = os.Stderr
                }()

                // Isolated temp directory
                tmpDir := t.TempDir()
                dataDir := filepath.Join(tmpDir, ".aether", "data")
                os.MkdirAll(dataDir, 0755)

                origRoot := os.Getenv("AETHER_ROOT")
                t.Setenv("AETHER_ROOT", tmpDir)
                defer os.Setenv("AETHER_ROOT", origRoot)

                // Create store
                s, err := createTestStore(dataDir)
                if err != nil {
                    t.Fatalf("failed to create test store: %v", err)
                }
                store = s

                // Catch panics
                defer func() {
                    if r := recover(); r != nil {
                        t.Errorf("PANIC: %v", r)
                    }
                }()

                rootCmd.SetArgs([]string{cmd.Name()})
                _ = rootCmd.Execute()

                // Verify some output was produced (help, error, or result)
                output := outBuf.String() + errBuf.String()
                if output == "" && !cmd.HasSubCommands() {
                    t.Logf("command produced no output (may be acceptable)")
                }
            })
        }
    }
    ```

    Handle edge cases:
    - Commands with subcommands (e.g., "help") may not produce output on their own -- skip output check for HasSubCommands()
    - Commands that require specific flags may error -- that's acceptable, as long as no panic
    - The test should be FAST (<10 seconds for all commands). Each command runs with no args, which is the simplest invocation.
