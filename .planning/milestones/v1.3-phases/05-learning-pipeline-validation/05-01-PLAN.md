Create `cmd/trust_event_instinct_cmds.go` with 10 Cobra commands. Each command follows the exact same pattern as `cmd/learning.go`: PersistentPreRunE initializes the store, the RunE function validates required flags, calls the Go package, and outputs via outputOK/outputError.

**Trust scoring commands (3):**

1. `trust-calculate` --flags: `--source` (required), `--evidence` (required), `--days-since` (default 0)
   Calls `memory.Calculate(memory.TrustInput{SourceType: source, Evidence: evidence, DaysSince: days})`.
   Outputs: `{score, source_score, evidence_score, activity_score, tier, tier_index}`

2. `trust-decay` --flags: `--score` (required), `--days` (required)
   Calls `memory.Decay(score, days)`.
   Outputs: `{decayed, original, days}`

3. `trust-tier` --flags: `--score` (required)
   Calls `memory.Tier(score)`.
   Outputs: `{tier, tier_index, score}`

**Event bus commands (3):**

4. `event-publish` --flags: `--topic` (required), `--payload` (required), `--source` (default "system"), `--ttl` (default 30)
   Creates `events.NewBus(store, events.Config{DefaultTTL: ttl})`, then `bus.Publish(ctx, topic, json.RawMessage(payload), source)`.
   Outputs: `{id, topic, timestamp, expires_at}`

5. `event-cleanup` --flags: `--dry-run` (default false)
   Creates bus, calls `bus.Cleanup(ctx, dryRun)`.
   Outputs: `{removed, remaining, dry_run}`

6. `event-replay` --flags: `--topic` (required), `--limit` (default 50), `--since` (optional RFC3339 timestamp)
   Creates bus, calls `bus.Replay(ctx, topic, since, limit)`.
   Outputs: `{events: [...], count}`

**Instinct store commands (4):**

7. `instinct-create` --flags: `--trigger` (required), `--action` (required), `--confidence` (default 0.75), `--domain` (optional), `--source` (default "observation"), `--evidence` (optional)
   Loads `colony.InstinctsFile` from store. Checks for duplicate by trigger+action. If duplicate found: boost confidence by 0.05 (cap 1.0), increment application count, update last_applied. If new: generate UUID-based ID (format `inst_<timestamp>`), compute trust tier via `memory.Tier(confidence)`, create `InstinctEntry` with full provenance, append to file. Cap at 30 instincts (archive lowest-confidence if exceeded). Save file. Also append to COLONY_STATE.json `memory.instincts` array with simpler `Instinct` struct (id, trigger, action, confidence, status:"active", domain, source, evidence, tested:false, created_at).
   Outputs: `{id, trigger, action, confidence, trust_tier, status, duplicate, total_instincts}`

8. `instinct-read-trusted` --flags: `--min-score` (default 0.5), `--domain` (optional), `--limit` (default 20)
   Loads `colony.InstinctsFile` from store, filters: `.Archived == false && .TrustScore >= minScore`, optionally filters by domain, sorts by TrustScore descending, limits.
   Outputs: `{instincts: [...], count}`

9. `instinct-decay-all` --flags: `--days` (default 30), `--dry-run` (default false)
   Loads `colony.InstinctsFile`, for each non-archived entry: `memory.Decay(entry.TrustScore, days)`, updates TrustScore and TrustTier. Archives entries with decayed < 0.25. Saves if not dry-run.
   Outputs: `{processed, archived, days, dry_run}`

10. `instinct-archive` --flags: `--id` (required)
   Loads `colony.InstinctsFile`, finds entry by ID, sets `.Archived = true`. Saves.
   Outputs: `{archived: id}`

All commands use the `init()` function pattern to register with `rootCmd.AddCommand(...)`. Each command's Use field matches the shell subcommand name exactly (e.g., `Use: "trust-calculate"`).

Create `cmd/graph_consolidation_cmds.go` with 6 Cobra commands.

**Graph commands (4):**

1. `graph-link` --flags: `--source` (required), `--target` (required), `--relationship` (required), `--weight` (default 1.0)
   Loads graph from store (`graph.NewGraph().LoadFromStore(store, "instinct-graph.json")`), calls `g.AddEdge(source, target, EdgeType(relationship), weight, timestamp)`, saves back.
   Outputs: `{edge_id, source, target, relationship, status}`

2. `graph-neighbors` --flags: `--node` (required), `--direction` (default "both"), `--relationship` (optional filter), `--min-weight` (default 0)
   Loads graph, calls `g.Neighbors(node, direction, filterRel, minWeight)`.
   Outputs: `{neighbors: [...], count}`

3. `graph-reach` --flags: `--node` (required), `--max-hops` (default 3), `--min-weight` (default 0)
   Loads graph, calls `g.Reach(node, maxHops, minWeight)`.
   Outputs: `{reachable: [...], count, hops_searched}`

4. `graph-cluster` --flags: (none)
   Loads graph, calls `g.DetectCycles()`. Returns cycle information.
   Outputs: `{cycles: [...], count}`

**Consolidation commands (2):**

5. `consolidation-phase-end` --flags: `--dry-run` (default false)
   Creates `memory.NewConsolidationService(store, events.NewBus(store, events.DefaultConfig()), queenPath, colonyName)` where queenPath is `store.BasePath() + "/QUEEN.md"` and colonyName is "unknown".
   Calls `service.Run(ctx)`.
   Outputs: `{type: "phase_end", instincts_decayed, instincts_archived, observations_decayed, promotion_candidates, queen_eligible}`

6. `consolidation-seal` --flags: `--dry-run` (default false)
   Runs 5 sequential steps (matching shell `_consolidation_seal`):
   Step 1: Run curation orchestrator via `curation.NewOrchestrator(store, bus).Run(ctx, dryRun)`
   Step 2: Run instinct-decay-all (load instincts, apply decay, save)
   Step 3: Run archivist (load instincts, archive those with confidence < 0.3)
   Step 4: Publish event via bus.Publish with topic "consolidation.seal"
   Step 5: Generate scribe report (write summary to store)
   Outputs: `{type: "seal", steps: [...], event_published, dry_run}`

All commands use the `init()` function pattern. The graph file path is always `"instinct-graph.json"`. The timestamp format is always `time.Now().UTC().Format("2006-01-02T15:04:05Z")`.
