## What Is It? (Full Body)

Aether is an open-source multi-agent framework modeled on ant colonies. Instead of micromanaging a single AI assistant, you launch a colony of 24 specialized workers -- builders, watchers, scouts, architects, and more -- that self-organize around your goals.

**How it works:**

You describe what you want to build. Aether breaks it into phases, assigns workers by caste, and coordinates them using pheromone signals -- the same way real ant colonies communicate. No prompt engineering, no babysitting individual agents. You give the colony a goal and it figures out the rest.

**What makes it different:**

- **Pheromone signals over prompt engineering.** Instead of writing elaborate prompts, you set FOCUS, REDIRECT, and FEEDBACK signals. The colony adapts its behavior based on what it learns, just like real ants adjusting their trails.
- **Memory that compounds.** Aether doesn't start from zero every session. It builds instincts from patterns, graduates those to wisdom, and eventually feeds a "Hive Brain" that makes the entire colony smarter over time.
- **Autopilot mode.** Set a goal, go to sleep, wake up to working software. The colony plans, builds, tests, and self-corrects.
- **28 built-in skills.** From code archaeology (git history analysis) to chaos testing to philosophical dreaming. Each worker has a real job, not a vague title.

**Built in Go. MIT licensed. Fully open source.**

Install with one command: `go install github.com/calcosmic/Aether@latest`
